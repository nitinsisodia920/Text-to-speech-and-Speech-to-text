## üêõ Troubleshooting

### Speech to Text not working?
- Check if your browser supports the feature (Chrome/Edge recommended)
- Grant microphone permissions when prompted
- Check your system's microphone settings
- Try refreshing the page

### No voices appearing in dropdown?
- Wait a few seconds for voices to load
- Try refreshing the page
- Some browsers load voices asynchronously

### Text to Speech not playing?
- Ensure your system volume is up
- Check browser sound permissions
- Try a different voice from the dropdown
